## NOTES/REMINDER
Notes ad reminders app keep notes and set reminders.

# Pages Discription
login page: user login/sign in (name, username, email, password)
Notes: User Notes (add/delete/edit notes)
Reminder: User Reminders (add/delete/edit/activate reminder), set date and time
active reminders: active reminders (deactivate/view upcoming reminder) 
Dashboard: Total notes/reminders, total active reminders
